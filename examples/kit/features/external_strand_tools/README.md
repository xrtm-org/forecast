# External Strand Tools

**Script**: `run_external_strand_tools.py`

Demonstrates how to integrate "Strands"—specialized, isolated tool execution environments or remote services—into the agent's workflow.

## Usage

```bash
# From repository root
python3 examples/kit/features/external_strand_tools/run_external_strand_tools.py
```

## Concepts
- **Tool Isolation**: Running untrusted or complex tools in a safe sandbox.
- **Remote Execution**: Calling tools hosted on other servers.
